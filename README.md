# Personal Expense Tracker
A web-based application to track daily expenses, categorize spending, and analyze total expenditure.

## Features
- Add daily expenses
- Categorize expenses
- View all expenses
- Calculate total spending

## Tech Used
- HTML, CSS
- PHP
- MySQL

## Setup
1. Install XAMPP
2. Start Apache & MySQL
3. Place folder in htdocs
4. Import database.sql
5. Run:
   http://localhost:8080/expense_tracker/index.html
